Common questions about home security systems and alarm monitoring

How much do alarm systems cost?

For many people, the price of a home security system is one of the most important factors in deciding which alarm system to purchase. There are four basic elements to consider: upfront (installation) costs, ongoing (monthly) subscription costs, equipment costs and municipal fees. Be aware that many home insurance providers offer discounts (up to 20 percent) for homes with monitored alarm systems.

Upfront Cost:

Installation is either professional or do-it-yourself (DIY); some companies let you choose an installation method, while others offer only one option. Professional installation costs can run $100-$500+; DIY is free, but it requires more of the consumer’s time.

Subscription cost:

Subscription security systems – home alarms monitored 24/7 by outside companies – carry a monthly fee, usually in the $20-$60 range.
